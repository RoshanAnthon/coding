Sorting:

	Theory

	You should know:

		Algorithm	Time	Stable	In-place	Notes
		Bubble Sort	O(n²)	Yes	Yes	Mostly academic
		Selection Sort	O(n²)	No	Yes	Minimal swaps
		Insertion Sort	O(n²)	Yes	Yes	Good for small/nearly sorted
		Merge Sort	O(n log n)	Yes	No	Divide & conquer
		Quick Sort	Avg O(n log n)	No	Yes	Most important
		Heap Sort	O(n log n)	No	Yes	Heap based
		Counting Sort	O(n+k)	Yes	No	Integers/range
		Radix Sort	O(nk)	Yes	No	Digits/strings
		Bucket Sort	Avg O(n)	Depends	Depends	Distribution-based
